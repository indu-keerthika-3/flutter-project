## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/indu-keerthika-3/flutter-project.git
   cd flutter-project
   ```

2. Install dependencies:

   ```bash
   flutter pub get
   ```

3. Run the app:

   ```bash
   flutter run
   ```
